# Jeu de Morpion

Reproduction du jeu du morpion en javascript

## Languages utilisés
- Javascript

## Fonctionnalités

- Choix du nom des joueurs
- Choix d'un avatar pour les joueurs
- Jeu Fonctionnel
- Comptage des scores
- Rejouer une partie
- Reset les scores

## Amélioration possible

- Ajout de règle customisé
- Amélioration du design
